About The Developer

Hi! My name is Tim Williamson.

I am a Full Stack Web Developer, with over 3 years of experience.

Frontend

I have used Angular to build highly modular and componentised frontend applications, that are intuitive, responsive and highly accessible.

I believe in using good semantic html, to help build webpages that are intelligible to developers as well as being styling and being accessible for users with disabilities.

Backend

I have built RESTful services in Spring Boot, Node and Sinatra. I am used to building services as part of a microservice architechture, and integrating with APIs and databases.

I have deployed work through CICD pipelines onto docker images, using Jenkins, Kubernates and Openshift. I have also deployed services onto AWS EC2 instances using AWS's CodeBuild and CodePipeline.

Productivity and code quality

I am passionate about building exceptional quality code. I use Test Driven Development with unit and integration tests. The majority of my experience has been using object oriented design principles, building small but extensible classes each with a single responsibility. I use git to collaborate effectively with co-workers, utilising code reviews and pair programming.

Other interests
